# Sales & Revenue Analytics API

## Prerequisites 
   - Install Node.js
   - Install Mongodb

## Steps

0. Install Dependencies
   ```bash
      npm install
   ```
1. Import data:
   ```bash
   npm run seed
   ```

2. Build the TypeScript code:
   ```bash
   npm run build
   ```

3. Start the server:
   ```bash
   npm start
   ```

   For development with auto-restart:
   ```bash
   npm run dev
   ```

4. Access the GraphQL Playground at:
   ```
   http://localhost:4000/graphql
   ```
